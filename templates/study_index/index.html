欢迎使用 Study Intoclaw 系统

API 接口

认证相关

POST /api/v1/auth/register/ - 用户注册
POST /api/v1/auth/login/ - 用户登录
POST /api/v1/auth/refresh/ - 刷新 Token
POST /api/v1/auth/logout/ - 用户登出

用户相关

GET /api/v1/users/me/ - 获取当前用户信息
PUT /api/v1/users/me/ - 更新用户信息
POST /api/v1/users/change-password/ - 修改密码

课程相关

GET /api/v1/courses/ - 获取课程列表
GET /api/v1/courses/{slug}/ - 获取课程详情
GET /api/v1/courses/{slug}/lessons/ - 获取课程章节
POST /api/v1/courses/{slug}/reviews/ - 提交课程评价
GET /api/v1/courses/{slug}/reviews/ - 获取课程评价

学习相关

GET /api/v1/learning/my-courses/ - 获取我的课程
GET /api/v1/learning/courses/{course_id}/progress/ - 获取课程进度
POST /api/v1/learning/lessons/{lesson_id}/progress/ - 提交章节进度

订单相关

POST /api/v1/orders/ - 创建订单
GET /api/v1/orders/ - 获取订单列表
GET /api/v1/orders/{order_no}/ - 获取订单详情
POST /api/v1/orders/{order_no}/cancel/ - 取消订单

支付相关

POST /api/v1/payments/ - 创建支付单（需要订单号）
GET /api/v1/payments/{payment_no}/ - 获取支付详情
POST /api/v1/payments/callback/wechat/ - 微信支付回调
POST /api/v1/payments/callback/alipay/ - 支付宝支付回调

退款相关

POST /api/v1/refunds/ - 申请退款（需要订单号、支付单号、金额）
